# MLOps Project - Breast Cancer Prediction

## Live Demo
- Frontend: https://mlops-vision.lovable.app
- Model Accuracy: 96.49%

## Tech Stack
- Data Lineage: OpenLineage + Marquez
- Versioning: DVC
- Data Quality: Great Expectations
- Feature Store: Feast
- Experiment Tracking: MLflow
- Orchestration: Docker + Kubernetes
- Deployment: FastAPI + ReactJS
- Monitoring: Grafana + Prometheus
- CI: GitHub Actions
- CD: ArgoCD
- Prompt Testing: Promptfoo

## Dataset
Breast Cancer Wisconsin dataset from sklearn
569 samples, 30 features, 96.49% accuracy
